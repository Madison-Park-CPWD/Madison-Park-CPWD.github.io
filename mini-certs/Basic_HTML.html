Master HTML Certification Requirements

To earn the Master HTML Certification, students must:

  • Complete ALL 6 mini-certificates
  • Capstone Project: Build a complete, multi-page website that demonstrates proficiency in all areas
  • Peer Review: Successfully review and provide feedback on 2 classmates' HTML projects
  • Teaching Moment: Lead a mini-lesson or tutorial for other students on any HTML topic

Assessment Notes for Teachers

Evaluation Criteria:

  • Functionality: Code works as intended
  • Standards Compliance: Proper HTML5 structure and semantics
  • Accessibility: Appropriate use of alt text, labels, and semantic elements
  • Code Quality: Clean, well-indented, readable code
  • Creativity: Thoughtful application of skills to solve real problems

Flexible Assessment Options:

  • Students can demonstrate proficiency through multiple pathways
  • Projects can be individual or collaborative
  • Allow resubmission after feedback for improvement
  • Consider real-world applications and student interests

Documentation:

  • Students should maintain a portfolio of their work
  • Encourage reflection on learning and problem-solving process
  • Include peer feedback and self-assessment components
